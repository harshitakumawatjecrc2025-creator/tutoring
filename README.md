# Free Tutoring & Mentoring Mobile App

A comprehensive Android application for connecting tutors and students for free tutoring and mentoring services.

## Features

- User authentication (Student/Tutor)
- Browse available tutors
- Book tutoring sessions
- Video chat integration
- Real-time messaging
- Session scheduling
- Rating and reviews
- Profile management

## Setup Instructions

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle files
4. Run on emulator or physical device

## Requirements

- Android SDK 21+
- Java 8+
- Android Studio 4.0+
